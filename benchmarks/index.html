How to read this page

LiveBench reports category scores on objective tasks. Arena reports blind human-preference Elo. SWE-bench Verified reports the rate at which an agent configuration resolves real repository issues.

Important limits

Scores from different benchmarks cannot be added together. The everyday-life view uses a preference proxy because no general authoritative benchmark covers that scenario.
