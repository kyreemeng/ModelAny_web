多模型启动器

一条提示词，比较多个模型

不必在多个模型标签页之间反复复制。ModelAny 会在你选择的模型中打开对话并填入同一提示词，让你更快比较不同视角。

为什么选择 ModelAny

少一些复制粘贴，多一些可靠比较。

一次输入，多个模型

输入一次，即可发给你选择的全部模型，不必再在多个标签页间粘贴。

更快比较不同视角

同一问题，多种模型思路并列呈现，一轮就能看到更多答案。

自动打开模型对话

自动为每个选中的模型打开独立对话页面并填入问题。

可选自动发送

可让 ModelAny 自动提交，也可只填入文本，发送前由你确认。控制权始终在你。

对话井然有序

同一轮打开的标签页自动归组，多模型对话井然有序。

本地优先的隐私

草稿、设置与记录保存在浏览器本地，不会上传到 ModelAny 服务器。

使用方式

三步，获得更多视角。

1

写下你的问题

打开 ModelAny 弹窗，输入要同时发送的问题，最多 5000 字。

2

选择要使用的模型

从八个支持的模型中选择一个或多个，可按需开启自动发送。

3

比较更好的答案

ModelAny 打开每个模型对话页，填入问题并可选发送，便于横向比较。

常见问题

ModelAny 是什么？

ModelAny 是一款免费开源的 Chrome 扩展，输入一次提示词即可同时发送给多个模型，方便横向比较回答。

ModelAny 支持哪些模型？

支持 8 个模型：ChatGPT、Gemini、DeepSeek、Kimi、GLM、通义千问、豆包、文心。

ModelAny 免费吗？

ModelAny 完全免费且开源，可从 Chrome 网上应用店安装，并在 GitHub 查看代码。

ModelAny 会保存我的提示词吗？

采用本地优先设计，草稿、设置与历史保存在浏览器本地，提示词只发送到你主动选择的模型，不会上传到 ModelAny 服务器。

如何安装 ModelAny？

打开 ModelAny 的 Chrome 网上应用店页面，点击“添加至 Chrome”。Microsoft Edge 扩展仍在审核中。

可以用 ModelAny 比较模型的回答吗？

可以，这正是核心功能。ModelAny 为每个选中的模型打开独立标签页并填入提示词，可选自动发送，一轮即可比较多个回答。

ModelAny 会自动发送提示词吗？

可选。开启自动发送会自动提交；关闭则只填入文本，由你手动确认发送。
